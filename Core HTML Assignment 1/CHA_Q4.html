List of tags

  1. Bold Tag(<b>): It is used to make the text bold.
  2. Anchor Tag(<a>): It is used to create hyperlinks or hypertext reference.
  3. Img Tag(<img>): It is used to embed images.
  4. Superscript Tag(<sup>): It is used to make the text superscripted.
  5. Em Tag(<em>): It makes the test bold and also emphasizes it.
