1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    Makes the web page visually appealing
    • Separation: CSS separates content and design for easier maintenance.
    • Consistency: Ensures uniform styles for a polished look.
    • Responsiveness: Adapts to various devices for better usability.

  3. JS

    Primary Use Cases Key Features/Benefits
    JavaScript is primarily used for adding interactivity and dynamic behavior to websites.
    • Interactivity: Enables dynamic and engaging web experiences.
    • Cross-Browser: Works on all major browsers.
    • Versatile: Used for front-end and back-end development, with a rich ecosystem.
